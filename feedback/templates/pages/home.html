Seja Bem-Vindo

Este é um sistema de feedback para alunos e professores. Aqui você pode visualizar e editar feedbacks de disciplinas.

Objetivo: O objetivo deste sistema é fornecer uma plataforma para que alunos e professores possam compartilhar feedbacks sobre as disciplinas, promovendo um ambiente de aprendizado mais colaborativo e eficaz.

Funcionalidades: O sistema permite que os alunos visualizem feedbacks de disciplinas, editem feedbacks existentes e adicionem novos feedbacks.

Como usar: Para começar, faça login no sistema. Após o login, você poderá acessar as funcionalidades disponíveis. Se você não tiver uma conta, acesse a área de cadastro abaixo para criar a sua.

Cadastro: Para se cadastrar, clique no link abaixo e preencha o formulário com suas informações.
